# 书签使用说明

## 书签解决什么问题

书签用于把当前编辑位置保存成一个可快速跳回的目标。每个书签会记录文件、行号、列号和该行的一小段正文摘要。

## 添加书签

把光标放到要标记的位置，然后使用任一方式添加：

- 点击编辑器工具栏里的书签图标。
- 按 `Cmd+Shift+B`，Windows/Linux 使用 `Ctrl+Shift+B`。

添加成功后会同时出现这些反馈：

- 顶部状态文字显示 `已添加书签 文件名:行号:列号`。
- 顶部工具栏的“书签列表”按钮出现数量角标，表示当前一共保存了多少个书签。
- 如果当前光标所在行已有书签，编辑器工具栏里的“添加书签”按钮会高亮。
- Monaco 编辑器左侧 gutter 会在有书签的行显示红色菱形标记。

## 判断当前行是否已有书签

看编辑器工具栏里的“添加书签”按钮：

- 普通状态：当前行还没有书签。
- 高亮状态：当前行已有书签，鼠标悬停会显示当前行已有几个书签。

也可以看编辑器左侧 gutter：有红色菱形标记的行就是本文件中已保存书签的位置。

注意：书签实际记录的是“行号 + 列号”。同一行不同列可以存在多个书签，所以高亮提示按“当前行”汇总，书签列表中会显示具体的 `行号:列号`。

## 查看这个文件有哪些书签

点击顶部工具栏的书签图标，或按 `Cmd+B` / `Ctrl+B` 打开书签列表。

在书签弹窗中选择 `当前文件`：

- 标题会显示 `书签 · 当前文件 N`。
- 列表只显示当前文件里的书签。
- 每一项显示文件名、正文摘要和 `行号:列号`。

## 查看所有添加过的书签

打开书签列表后选择 `所有文件`：

- 标题会显示 `书签 · 所有文件 N`。
- 列表会显示所有文件和未保存标签页中的书签。
- 顶部搜索框可以按文件名、文件路径、正文片段或 `行号:列号` 搜索。

## 跳转和删除

在书签列表中：

- 单击一行：选中书签。
- 双击一行、按 `Enter`，或点击 `跳转`：跳到该书签位置。
- 点击行尾删除按钮：删除该书签。
- 列表获得焦点时，按 `Delete` 或 `Backspace`：删除当前选中的书签。
- 按 `Esc`：关闭书签列表。

键盘选择：

- `Up` / `Down`：上下移动选择。
- `Ctrl+Up` / `Ctrl+Down`：快速翻动选择。

## 使用限制

- 已保存文件中的书签可以在文件未打开时重新打开并跳转。
- 未保存标签页中的书签只能在该标签页仍然存在或会话恢复后跳转。
- 书签是位置记录，不会跟随正文语义移动；如果在书签前面大量增删内容，原来的行号可能指向新的内容。
